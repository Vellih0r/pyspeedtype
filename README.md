# How to install pyspeed

1. Clone this repository on your computer:
    `git clone https://github.com/Vellih0r/pyspeedtype`

2. Install python:
    `sudo apt install python3`

3. Install requirements:
    `cd pyspeedtype`
    `chmod +x required.sh`
    `./requiredsh`

4. Run CLI:
    `python3 pyspeed.py`
